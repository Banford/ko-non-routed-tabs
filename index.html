Knockout Non-Routed Tabs

Summary:

  • Does not use the bootstrap JavaScript or jQuery.
  • Each 'tab-content' panel is a knockout component.
  • Selected tab is an observable on the 'pb-ko-tabs' viewModel.
  • The viewModel for a tab will be constructed each time the tab is switched to.
  • The tabs have to use the containerless control flow comment binding syntax.
  • Uses $componentTemplateNodes to render the child nodes of 'pb-ko-tabs'.

Disabled Tabs

It's possible to add tabs and not specify a key for the component they are supposed to load. In this case the tabs will be rendered as disabled links and clicking them will not attempt to switch tab.
